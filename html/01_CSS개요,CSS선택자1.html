CSS(Cascading Style Sheets)

        마크업 언어(HTML)가 실제로 화면에 표시되는 방법을 기술하는 언어
        (모양, 색상, 크기, 배경이미지, 움직임, ....)

        하나의 요소에 여러 CSS 코드를 겹쳐서 원하는 스타일 만들어냄

        W3C에서 CSS를 웹 표준으로 지정함

CSS 선택자(CSS Selectoir)

        현재 문서 내에서 특정 요소를 선택하는 문자 또는 문자열(기호)

        * 요소(Element) : 시작태그 + 속성 + 내용 + 종료태그
    ** CSS 코드를 작성하는 방법 **
  • head 태그에 style 태그를 작성한 후 내부에 css 코드 작성
  • 원하는 요소에 직접 style 속성을 이용해서 css 코드 작성
  • 외부에 .css 파일을 생성하여 연결하는 방법

태그 선택자

        - 현재 문서 내에 특정 태그를 모두 찾아 선택하는 선택자

        [작성법]

        
            태그명 { css 코드; }
